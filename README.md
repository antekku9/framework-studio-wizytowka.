# Framework Studio – wizytówka QR

Gotowe repozytorium pod GitHub Pages.

## Publikacja
1. Utwórz nowe repozytorium na GitHubie.
2. Wrzuć pliki `index.html` oraz `vcard.vcf` do głównego katalogu repozytorium.
3. Wejdź w Settings → Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Po chwili strona będzie dostępna pod adresem GitHub Pages.

## QR
Kod QR wygeneruj do adresu opublikowanej strony GitHub Pages.
