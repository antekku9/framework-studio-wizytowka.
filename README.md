# Framework Studio - wizytówka online

Statyczna wizytówka online przygotowana pod GitHub Pages.

## Jak wrzucić na GitHuba

1. Utwórz nowe repozytorium na GitHubie, np. `framework-studio-wizytowka`.
2. Wrzuć do niego pliki z tej paczki:
   - `index.html`
   - `vcard.vcf`
   - `README.md`
3. Wejdź w `Settings` → `Pages`.
4. W sekcji `Build and deployment` ustaw:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Zapisz ustawienia.
6. Po chwili strona będzie dostępna pod adresem w stylu:
   `https://twoj-login.github.io/framework-studio-wizytowka/`

## QR code

Po opublikowaniu strony wygeneruj kod QR do adresu GitHub Pages. Osoba skanująca QR trafi bezpośrednio na wizytówkę.
